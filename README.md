# Classification Multi-classe Iris - Baseline

## Description
Mini-projet ML : prédire l'espèce de fleurs Iris à partir de 4 features (sepal/petal length/width).  

## Structure
- `src/train_baseline.py` : script d'entraînement et d'évaluation baseline Logistic Regression
- `data/` : datasets (non versionné)
- `models/` : modèles entraînés (non versionné)
- `notebooks/` : exploration et visualisation
- `reports/` : métriques et résultats

## Usage
```bash
python src/train_baseline.py
